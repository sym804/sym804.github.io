Project 05

Selector Extractor - QA 자동화용 크롬 확장

QA 자동화 작업 시 반복되는 셀렉터 수동 추출 작업을 제거하기 위해, 엘리먼트 클릭만으로 최적 셀렉터와 Playwright 코드를 자동 생성하는 크롬 확장 프로그램을 직접 기획·개발

  • 7가지 셀렉터 타입 우선순위 추출 (ID, data-testid, aria-label, role 등)
  • Playwright 6가지 액션 코드 자동 생성 (click, fill, check, select 등)
  • 실시간 말풍선 UI로 클릭 즉시 셀렉터 확인 + 클립보드 복사
  • 사이드 패널 히스토리 관리, 추출 이력 JSON/CSV 내보내기
  • 동적 클래스 필터링 (랜덤 해시/번호 자동 제거)
  • Manifest V3 기반, Chrome Store 배포 준비 완료
