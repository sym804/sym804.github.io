Why

왜 QA 엔지니어가 만들었나

AI Agent의 실수를 QA 현장에서 다수 발견, 멀티 Agent 교차 검증으로 해결합니다.

AI Agent의 한계
  • 할루시네이션 (없는 API·함수 호출)
  • 엣지 케이스·예외 처리 누락
  • 보안·인증 로직 간과
QA 현장에서 발견
  • 실제 QA 업무 중 AI 산출물 버그 다수 확인
  • 존재하지 않는 API 호출·스키마 불일치
  • 단일 모델의 관점 편향과 검증 공백
각 Agent 강점 활용
  • Claude Code: 설계와 코딩
  • Codex: 코드 리뷰와 테스트
  • Gemini CLI: 추론 및 검색
Agents

3개 AI 에이전트

각 에이전트가 고유한 강점을 발휘하며, 토론과 코딩에서 상호 보완적으로 협업합니다.

Claude
Sonnet

기획, 설계, 코드 작성을 담당하는 메인 에이전트. 웹 검색을 우선 활용하여 최신 정보 기반의 정확한 답변을 생성합니다.

기획/설계 코드 작성 웹 검색
Codex
o1

코드 리뷰, 테스트 전략 수립, 최종 보고를 담당. 깊은 추론 능력으로 코드의 품질과 안정성을 검증합니다.

코드 리뷰 테스트 전략 최종 보고
Gemini
flash-preview + fallback

창의적 관점과 빠른 응답 속도를 제공. flash-preview 모델 기본 사용, 실패 시 자동 fallback으로 안정성을 확보합니다.

창의적 관점 빠른 응답 Fallback
Modes

3가지 운영 모드

상황에 맞는 최적의 협업 방식을 선택하여 문제를 해결합니다.

토론 모드

3개 AI가 자유롭게 토론하며 최적의 답을 도출합니다.

  • 라운드 기반 자유 토론 (최대 10라운드)
  • 합의 감지: 전원(3/3) 즉시 종료, 다수(2/3) + 교착 시 종료
  • 합의 후 통합 답변 자동 생성
  • 에이전트 오류/타임아웃 시 백업 에이전트 자동 대체
  • 웹 검색 강제로 실시간 수치 환각 방지
코딩 모드

5단계 파이프라인으로 고품질 코드를 생산합니다.

1 Claude가 코드 작성
2 Codex가 코드 리뷰
3 3개 에이전트 동시 테스트
4 이슈 수정 루프 (최대 3회)
5 Codex 최종 보고
  • 경로 화이트리스트 기반 보안 적용
브릿지 모드

Slack 메시지를 로컬 CLI에 직접 연결합니다.

  • 특정 프로젝트 폴더의 Claude/Codex CLI에 연동
  • --continue 옵션으로 기존 대화 이어가기
  • Slack에서 직접 로컬 개발 환경 제어
Debate Example

실제 토론 예시

사용자 "비개발자가 프로그래밍을 하나 배운다면?"
Claude Python 추천. 문법이 직관적이고 자동화/데이터 분석에 바로 활용 가능
Codex JavaScript 추천. 웹 프론트/백엔드 둘 다 가능하고 결과물이 눈에 보여 동기부여

합의(2/3) Python - 직관적 문법, 자동화/데이터 분석에 즉시 활용

Key Features

핵심 기능

안정성, 보안, 성능을 위해 설계된 엔지니어링 기능들입니다.

비동기 병렬 처리
asyncio 기반으로 3개 AI 에이전트를 동시에 호출하여 응답 대기 시간을 최소화합니다.
타임아웃 + 백업 에이전트
Claude 실패 시 Codex-B(🟤)로, Codex/Gemini 실패 시 Claude-B(🟡)로 자동 전환하여 서비스 연속성을 보장합니다.
보안
경로 화이트리스트로 허용된 디렉토리만 접근 가능. 환경변수 토큰 필터링으로 민감 정보 노출을 차단합니다.
출력 정제
Claude JSON artifact, Codex 헤더, Gemini xterm 노이즈 등 각 에이전트별 불필요한 출력을 자동 제거합니다.
진행률 표시
15초마다 Slack 메시지를 업데이트하여 현재 처리 상태를 실시간으로 사용자에게 전달합니다.
취소 시스템
!stop 명령어로 진행 중인 작업을 즉시 중단. 실행 중인 모든 서브프로세스를 안전하게 종료합니다.
Gemini Rate-limit 핸들링
Semaphore 동시성 제어, 5분 쿨다운, 모델 fallback을 조합하여 API 제한을 자동으로 우회합니다.
CLI 헬스체크
부팅 시 Claude, Codex, Gemini CLI의 정상 작동 여부를 자동으로 점검합니다.
