Why this tool?

반복적인 셀렉터 탐색,
자동화할 수 있습니다

QA 엔지니어가 테스트 자동화 스크립트를 작성할 때 매번 DevTools를 열고 Elements 탭에서 적절한 셀렉터를 수동으로 찾는 과정은 비효율적입니다.

ID가 있는지 확인하고, data-testid가 있는지 살피고, 유니크한 CSS 셀렉터를 조합하고 - 이 작업을 엘리먼트 하나마다 반복해야 합니다.

Selector Extractor는 클릭 한 번으로 7가지 타입의 셀렉터를 자동 추출하고, 유니크 셀렉터를 우선 정렬하며, Playwright 코드까지 생성합니다.

Before: 기존 방식
F12로 DevTools 열기
Elements 탭에서 요소 탐색
속성 하나씩 확인 (ID, class, data-*)
유니크한지 Console에서 수동 검증
Playwright 코드 직접 작성
After: Selector Extractor
요소 클릭 한 번
7가지 셀렉터 자동 추출 + 유니크 검증
Playwright 코드 원클릭 복사
Core Features

QA 엔지니어를 위한
핵심 기능 7가지

스마트 셀렉터 추출

ID > data-testid > 유니크 클래스 > CSS > XPath > ARIA Role > Text 7가지 타입을 자동 추출하고, 유니크 셀렉터를 우선 정렬합니다.

querySelectorAll uniqueness check priority sort

실시간 말풍선 UI

호버 시 블루 아웃라인으로 요소를 강조하고, 클릭 시 그린 하이라이트와 함께 UNIQUE/매치 수 뱃지를 표시합니다.

hover highlight click feedback badge

Playwright 코드 생성

locator(), get_by_role(), get_by_test_id(), get_by_text()와 click/fill/check/hover/visible/text assert 6가지 액션 코드를 자동 생성합니다.

locator() get_by_role() 6 actions

사이드 패널 히스토리

추출한 셀렉터를 누적 저장하고, 실시간 검색 기능과 페이지별/시간별 통계를 제공합니다.

chrome.sidePanel search statistics

데이터 내보내기

JSON/CSV 형식으로 내보내기를 지원하며, 날짜별 파일명을 자동으로 생성합니다.

JSON CSV auto filename

동적 클래스 필터링

css-*, se-*, 해시 기반 자동생성 클래스를 자동으로 제외하여 안정적인 셀렉터만 추출합니다.

regex filter hash detection stable selector

클립보드 복사

원클릭으로 셀렉터와 Playwright 코드를 복사하고, 복사 완료 시 시각적 피드백을 제공합니다.

navigator.clipboard visual feedback
